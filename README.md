# BarberTurn App

## Project Overview
BarberTurn is an innovative application designed to streamline the process of booking appointments at barber shops. It provides users with a platform to find, book, and manage appointments with barbers in their area, enhancing the overall user experience.

## Installation
To install BarberTurn App, follow these steps:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Ricardoarangob26/BarberTurn-App.git
   ```
2. **Navigate to the project directory**:
   ```bash
   cd BarberTurn-App
   ```
3. **Install dependencies**:
   ```bash
   npm install
   ```
4. **Run the application**:
   ```bash
   npm start
   ```

## Architecture
BarberTurn is built using a modern web architecture that includes:
- **Frontend**: Developed using React.js, providing a dynamic and responsive user interface.
- **Backend**: Node.js and Express.js are used to handle server-side logic and API requests.
- **Database**: MongoDB is utilized for storing user profiles, barbers, appointments, and other necessary data.

The application follows a microservices architecture, allowing for better scalability and maintainability.

## Features
- **User Registration/Login**: Secure user authentication, enabling users to register and log into the application.
- **Barber Search**: Users can search for barbers based on location, ratings, and services offered.
- **Appointment Booking**: Allows users to book appointments with their chosen barber, including the option to select specific services and time slots.
- **User Profiles**: Each user can create and manage their profile, including their bookings and personal information.
- **Reviews and Ratings**: Users can leave reviews and rate barbers based on their experiences, helping others make informed decisions.
